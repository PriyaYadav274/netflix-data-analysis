# Netflix Data Analysis

## Objectives
- Analyze the Netflix dataset to understand trends in movies and TV shows.
- Perform data cleaning and exploratory data analysis (EDA).
- Identify patterns in genres, release years, and content distribution.
- Visualize insights using Python data visualization libraries.

---

## Dataset
Dataset used: `netflix_titles.csv`

The dataset contains information about Netflix movies and TV shows, including:
- Title
- Type (Movie or TV Show)
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre (Listed In)
- Description

---

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Key Insights
- Movies make up the majority of Netflix content compared to TV shows.
- Netflix content production increased significantly after 2015.
- Some genres appear more frequently than others on the platform.
- Netflix content is distributed across multiple countries worldwide.
